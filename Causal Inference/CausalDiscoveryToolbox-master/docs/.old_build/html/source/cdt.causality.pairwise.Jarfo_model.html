cdt.causality.pairwise.Jarfo_model.hsic module¶

HSIC independence test

cdt.causality.pairwise.Jarfo_model.hsic.FastHsicTestGamma(X, Y, sig=[-1, -1], maxpnt=200)[source]¶

This function implements the HSIC independence test using a Gamma approximation to the test threshold Inputs: X contains dx columns, m rows. Each row is an i.i.d sample Y contains dy columns, m rows. Each row is an i.i.d sample sig[0] is kernel size for x (set to median distance if -1) sig[1] is kernel size for y (set to median distance if -1) Outputs: testStat: test statistic

Use at most maxpnt points to save time.
